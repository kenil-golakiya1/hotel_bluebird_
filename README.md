# hotel-bluebird-
 💡 Overview

The system provides two main interfaces:

Guest Panel: Allows users to register, log in, browse available rooms, view details, and book rooms online.

Admin Panel: Enables hotel staff to manage rooms, monitor bookings, handle guest data, generate reports, and respond to guest queries.

By replacing manual record-keeping with an automated digital platform, this project improves accuracy, saves time, and enhances customer satisfaction.

⚙️ Features

Guest Features

User registration and login

Real-time room availability

Online room booking with confirmation

Gallery and facilities view

Send messages or queries to hotel management

Admin Features

Secure login and authentication

Room and booking management (CRUD operations)

User management and query handling

Payment tracking and digital receipts

Automated reports for bookings, occupancy, and revenue

System Features

Centralized MySQL database

Data accuracy and security

Responsive and user-friendly interface

Scalable design for future enhancements

🧩 Technologies Used

Frontend: HTML, CSS, JavaScript, Bootstrap

Backend: PHP

Database: MySQL

Server: XAMPP (Apache + MySQL)

🚀 Future Enhancements

Integration of online payment gateway

Advanced reporting with charts and analytics

Auto-email notifications for booking confirmations
